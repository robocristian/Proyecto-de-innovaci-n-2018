"Grabadora en placa de cobre"

Con este proyecto buscamos resolver el metodo tradicional de imprimir un circuito en una placa de cobre ahorrando tiempo.

El objetivo de esta página web es el de presentar nuestro proyecto, plantear el fin de este, identificar los problemas que resolveria, y para que mercado esta destinado..
